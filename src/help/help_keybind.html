キーボードショートカット早見表

※Macの場合、CtrlはCommandに、AltはOptionに変わります。
ショートカットコマンド 機能
F1 ヘルプを表示
n いまなにしてるテキストエリアにフォーカス
b CWテキストボックスにフォーカス
Ctrl+Del 直前の投稿を削除
Ctrl+z 直前の投稿を削除して内容をコピー
Ctrl+v 直前の投稿の内容をコピー
Ctrl+i 直前の投稿につなげる
Ctrl+F5 画面全体を再読込する
Ctrl+f 検索ウィンドウを表示する
F4 リプライ投稿ウィンドウ表示、投稿詳細表示をしているときに表示ウィンドウを閉じる
a, ← カラムのカーソルを左に移動
d, → カラムのカーソルを右に移動
Shift+a, ← カーソルのカラムを閉じて左のカラムを開いてカーソルを合わせる
Shift+d, → カーソルのカラムを閉じて右のカラムを開いてカーソルを合わせる
Ctrl+Shift+a, ← 左のカラムを開いてカーソルを合わせる
Ctrl+Shift+d, → 右のカラムを開いてカーソルを合わせる
Enter カラムのカーソルを下に移動
Shift+Enter カラムのカーソルを上に移動
w, ↑ カーソルのカラムを上にスクロール
s, ↓ カーソルのカラムを下にスクロール
Shift+w, ↑ カーソルのカラムを上に多めにスクロール
Shift+s, ↓ カーソルのカラムを下に多めにスクロール
Ctrl+w, ↑ カーソルのカラムをトップに移動
F5 カーソルのカラムをリロードする
f カーソルのカラムの可変幅ON/OFFを切り替える
1～9(テンキーも可) 対応する番号のカラムを開閉する
(開いた場合はそのカラムにカーソルを合わせる)
(投稿フォーム)
Ctrl+Enter		 投稿する
(投稿フォーム)
Shift+Enter		 投稿して投稿フォームのフォーカスを維持する
(投稿フォーム)
Alt+↑,↓		 投稿先アカウントを変更する
(投稿フォーム)
Alt+←,→		 公開範囲を変更する
(投稿フォーム)
Esc		 投稿フォームからフォーカスアウトする
